# Simulated-Poverty-Gap-Index-Reduction-Project
Utilizing RStudio to depict the Poverty Gap Index Reduction with Data Visualization and Analysis

# R Project Simulated Poverty GAP Index Reduction 

#Author: Nick Winslow

#This project measures Simulated poverty gap index reduction measures, and how the estimated decrease in poverty gap attributable to a social protection and labor (SPL) program. Under the assumption that household income or consumption would fall by the full value of the transfer in the absence of the program.

+ 'data/' raw & processed data
+ 'scripts/' contains workflow files of the project
+ 'R/' contains function utilities to process, visualize and analyze the data set

To process the project, source 'Sim. Pov. Scripts/start.R'
